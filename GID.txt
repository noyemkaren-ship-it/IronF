🚀 IronF Framework — Документация

IronF — это мега-простой фреймворк для создания красивых сайтов. Забудь про сложный CSS и кучу файлов — тут всё в одном месте и на человеческом языке!

📦 Установка и запуск

Скачай IronF
Создай файл main.html в корне проекта
Напиши в нём код на IronF
Запусти компилятор
Открой src/index.html в браузере
Радуйся жизни! 🎉
📝 Базовая структура

Каждый проект IronF состоит из трёх секций:

text
css          ← тут стили (или команда basic для готовых)
html         ← тут HTML-код
endhtml      ← закрываем HTML (ОБЯЗАТЕЛЬНО!)
jscript      ← тут JavaScript
Минимальный рабочий пример:

text
css
basic
html
meta МойСайт
<h1>Привет, мир!</h1>
<p>Это мой первый сайт на IronF</p>
endhtml
jscript
alert("JavaScript работает!");
🎨 Секция CSS

Команда basic — Готовый дизайн за 1 слово

Хочешь красивый сайт без возни с CSS? Просто напиши:

text
css
basic
И всё! Твой сайт получит:

🍏 Красивые системные шрифты (Apple, Windows, Android)
📱 Адаптивность (хорошо выглядит на телефонах)
🎨 Стильные кнопки и поля ввода
✨ Анимации при наведении
📐 Автоматические отступы и центрирование
Свой CSS — Для тех, кто хочет больше

Можешь писать обычный CSS:

text
css
a {
    color: red;
    text-align: center;
}
🎯 Сокращения IronF (фишки для ленивых!)

IronF понимает сокращённые команды для самых частых CSS-свойств:

Команда IronF	Полный CSS	Что делает
talign:	text-align:	Выравнивание текста
fsize:	font-size:	Размер шрифта
Пример:

text
css
a {
    fsize: 50px;
    talign: center;
    color: blue;
}
Важно:

После значения нужна ;
Можно смешивать сокращения и обычный CSS
Значения для talign:

center — по центру
left — слева (или start)
right — справа (или end)
Значения для fsize:

16px, 20px, 50px и т.д.
1.5em, 2rem
small, medium, large
🏗️ Секция HTML

Команда meta — Заголовок страницы

text
html
meta НазваниеВкладки
Обязательно пиши meta перед HTML-кодом! Это создаст всю структуру документа за тебя:

DOCTYPE
Кодировку UTF-8
Адаптивный viewport
Подключение CSS и JS
Заголовок вкладки браузера
HTML-код

После meta пиши обычный HTML:

text
html
meta МойБлог
<h1>Добро пожаловать!</h1>
<p>Это параграф текста</p>
<button>Нажми меня</button>
<input type="text" placeholder="Введи имя">
Закрытие HTML

text
endhtml
ОБЯЗАТЕЛЬНО! Без этой команды компилятор выдаст ошибку!

⚡ Секция JScript

Пиши JavaScript-код после команды jscript:

text
jscript
alert("Привет!");
console.log("Сайт загружен!");

document.querySelector("button").onclick = function() {
    alert("Кнопка нажата!");
};
🚨 Обработка ошибок

IronF скажет тебе, если ты накосячил:

Ошибка	Сообщение
Забыл endhtml	🚨 НАЙДЕНА ОШИБКА: В КОДЕ НЕТ endhtml, А ОН ОБЯЗАТЕЛЕН!
Скоро будет больше проверок — компилятор станет твоим строгим, но любящим наставником! 👨‍🏫

Защита от дурака — встроенные проверки ошибок
