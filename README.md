🚀 IronF Framework

Пиши сайты на человеческом языке

IronF — это микро-фреймворк для тех, кто устал от трех файлов, сложного CSS и вечной возни с подключениями. Один файл. Понятные команды. Красивый результат.

📦 Установка

Скопируй проект к себе на компьютер:

git clone https://github.com/noyemkaren-ship-it/IronF.git
cd IronF

Убедись, что у тебя установлен Python 3.6 или новее:

python --version

Если Python не установлен — скачай его с python.org и установи.

🎮 Быстрый старт за 30 секунд

Открой блокнот или любой редактор кода и напиши:

📝 Как устроен файл main.html

Твой файл всегда состоит из трёх частей, и они идут строго по порядку:

css ← тут стили (как выглядит)
html ← тут содержимое (что показываем)
jscript ← тут логика (что делает)

Каждая секция начинается со своего слова-заголовка. Давай разберём каждую подробно.

📌 Секция CSS — настройка внешнего вида

CSS отвечает за красоту: цвета, шрифты, отступы, анимации. В IronF есть 4 способа работать со стилями.

Способ 1: Готовый дизайн Basic (для старта)

Просто напиши:

css
basic

Команда basic сама сделает всё красиво:

Шрифты будут удобными и крупными
Кнопки и поля ввода станут приятными на вид
Всё будет выровнено по центру
Появятся плавные анимации при наведении

Это лучший выбор, если ты только начинаешь!

Способ 2: Обычный CSS

Пиши CSS так, как ты его знаешь:

css
body {
background-color: #f0f0f0;
}
h1 {
color: blue;
fsize: 28px;
}
a {
color: red;
text-decoration: none;
}

Способ 3: Сокращения IronF

Чтобы писать меньше букв, используй короткие команды:

css
talign: center;
fsize: 24px;
color: blue;

Сокращение Что оно означает
talign: выравнивание текста (по центру, по левому краю, по правому)
fsize: размер шрифта (в пикселях)
color: цвет текста

Способ 4: Импорт CSS из другого файла

Если у тебя уже есть готовый CSS-файл, просто подключи его:

css
import styles/theme.css
import common/buttons.css

Путь к файлу пишется через пробел после слова import.

💡 Ты можешь смешивать все 4 способа! Например:

css
basic
import my-colors.css
h1 {
talign: left;
}
talign: center;

📌 Секция HTML — содержимое страницы

HTML — это то, что видит пользователь: заголовки, текст, кнопки, картинки.

Обязательная команда meta

Секция HTML всегда начинается с команды meta, после которой ты пишешь название вкладки браузера:

html
meta МойКрутойСайт
... тут остальное содержимое ...
endhtml

meta сама создаёт всю невидимую структуру страницы, тебе не нужно о ней думать.

Обязательная команда endhtml

В конце секции html обязательно напиши endhtml. Если забудешь — компилятор выдаст ошибку и подскажет, что делать.

html
meta Заголовок

<h1>Привет</h1> <p>Текст</p> endhtml ← не забудь!
📖 Гайд по HTML-тегам

Вот основные теги, которые ты можешь использовать. Не бойся — их не нужно учить все сразу, просто держи под рукой как шпаргалку.

Заголовки

Заголовки делают текст крупным и жирным. Бывают от h1 (самый большой) до h6 (самый маленький).

<h1>Самый большой заголовок</h1> <h2>Заголовок поменьше</h2> <h3>Ещё меньше</h3> <h4>Средний заголовок</h4> <h5>Маленький заголовок</h5> <h6>Самый маленький заголовок</h6>
Обычно на странице используют h1 для главного названия и h2 или h3 для разделов.

Абзацы (параграфы)

Обычный текст пишется в теге <p>:

<p>Это абзац с текстом. Здесь можно писать много-много слов, и они будут выглядеть как обычный текст на странице.</p> <p>А это второй абзац. Между абзацами автоматически появляется отступ.</p>
Кнопки (старый способ)

Можно создать кнопку и потом привязать к ней действие:

<button id="myBtn">Нажми меня</button>

Потом в секции jscript ты сможешь написать, что произойдёт при нажатии. Но в IronF есть способ проще — смотри раздел "Быстрые кнопки" ниже.

Поля ввода

Чтобы пользователь мог что-то напечатать:

<input type="text" id="name" placeholder="Введи своё имя">
type="text" — это текстовое поле
id="name" — уникальное имя, по которому ты будешь обращаться к полю из jscript
placeholder="..." — подсказка внутри поля, которая исчезает при вводе

Ещё бывают другие типы полей:

<input type="password" id="pass" placeholder="Пароль"> <input type="email" id="mail" placeholder="Твоя почта"> <input type="number" id="age" placeholder="Возраст">
Ссылки

Ссылка на другой сайт или страницу:

<a href="https://google.com">Перейти в Google</a>
<a href="/about.html">О нас</a>

Картинки

<img src="photo.jpg" alt="Моё фото">
src — путь к файлу картинки
alt — текст, который покажется, если картинка не загрузилась

Списки

Маркированный список (с точками):

<ul> <li>Первый пункт</li> <li>Второй пункт</li> <li>Третий пункт</li> </ul>
Нумерованный список (с цифрами):

<ol> <li>Шаг первый</li> <li>Шаг второй</li> <li>Шаг третий</li> </ol>
Выделение текста

<b>Жирный текст</b> — делает текст жирным
<i>Курсив</i> — делает текст наклонным
<u>Подчёркнутый</u> — подчёркивает текст

Пример:

<p>Это <b>очень важное</b> сообщение, обратите <i>внимание</i>!</p>
Перенос строки

Если нужно перейти на новую строку внутри абзаца:

<p>Первая строка<br>Вторая строка</p>
Тег
не нужно закрывать.

Таблицы

Простая таблица:

<table border="1"> <tr> <th>Имя</th> <th>Возраст</th> </tr> <tr> <td>Иван</td> <td>25</td> </tr> <tr> <td>Мария</td> <td>22</td> </tr> </table><tr> — строка таблицы <th> — заголовок столбца (жирный) <td> — обычная ячейка border="1" — показывает границы таблицы
Контейнер (div)

<div> — это невидимый контейнер, в который можно положить другие элементы:<div id="main-block"> <h2>Заголовок блока</h2> <p>Текст внутри блока</p> </div>
Пригодится, когда захочешь сгруппировать элементы или показать/спрятать их разом через jscript.

🔘 Быстрые кнопки (фишка IronF!)

В IronF можно создать кнопку с готовым действием всего за две строки! Это намного проще, чем писать <button> и потом настраивать её в скриптах.

Как это пишется:

html
meta МойСайт

<h1>Заголовок</h1> btn Нажми меня mySuperFunction() endhtml
Правила:

Первая строка: слово btn и через пробел текст, который будет написан на кнопке.
Вторая строка: имя твоей функции и обязательно скобки () на конце.

⚠️ Очень важно: скобки обязательны! Пиши startProcess(), а не startProcess. Без скобок кнопка не заработает.

Что происходит внутри:

IronF сам создаст красивую кнопку, даст ей уникальное имя и автоматически свяжет с твоей функцией. Когда пользователь нажмёт на кнопку — запустится твоя функция из секции jscript.

Полный рабочий пример:

css
basic

html
meta БыстрыеКнопки

<h1>Пример с кнопками</h1> <p id="status">Статус: пока ничего не делали</p> btn Запустить startProcess() btn Очистить resetAll() endhtml
jscript
function startProcess() {
txt("status", "Статус: процесс пошёл!");
print("Ура, процесс запущен!");
}

function resetAll() {
txt("status", "Статус: всё очищено");
print("Всё вернулось обратно!");
}

Скопируй этот пример, запусти компилятор и посмотри, как это работает!

🔗 Как принимать данные с бэкенда

Иногда твой сайт получает данные от сервера. Например: имя пользователя, список товаров, новости. Сервер вставляет эти данные в твой main.html ещё до того, как IronF его обработает.

Что ты видишь на своей стороне

В твоём файле main.html появляются специальные метки — переменные, которые выглядят как ${имя_переменной}. Твоя задача — просто использовать их в нужном месте.

Самый безопасный способ — спрятать данные в JSON

Представь, что сервер передал тебе переменные username, items и is_admin. Вот как правильно их принять и использовать:

В секции html создай скрытый контейнер для данных:

html
meta МойСайт

<script type="application/json" id="my-data"> { "username": "${username}", "items": ${items_json}, "is_admin": ${is_admin_js} } </script><h1>Привет, <span id="uname">гость</span>!</h1> <p id="status"></p> <ul id="my-list"></ul>
btn Показать права
showRights()
endhtml

В секции jscript достань эти данные и используй:

jscript
let data = {};

onReady(function() {
// Достаём данные из скрытого контейнера
let raw = gid("my-data").textContent;
data = JSON.parse(raw);

// Теперь data.username, data.items и data.is_admin доступны!

// Показываем имя пользователя
txt("uname", data.username);

// Проверяем, админ ли он
if(data.is_admin) {
txt("status", "Вы администратор!");
} else {
txt("status", "Вы обычный пользователь");
}

// Рисуем список
let listHtml = "";
for(let i = 0; i < data.items.length; i++) {
listHtml += "<li>" + data.items[i] + "</li>";
}
html("my-list", listHtml);
});

function showRights() {
if(data.is_admin) {
print(data.username + ", вам можно всё!");
} else {
print(data.username + ", у вас обычные права");
}
}

Что здесь происходит:

Сервер заранее вставил данные в ${...} метки
Мы спрятали их в теге <script type="application/json"> — это хитрый способ передать данные на страницу так, чтобы они не были видны обычному пользователю
В jscript мы эти данные читаем через gid("my-data").textContent
JSON.parse() превращает текст в удобный объект, с которым можно работать
Дальше используем данные как обычные переменные: показываем на странице, проверяем условия, рисуем списки
Простой способ (для одной переменной):

Если у тебя всего одна переменная, можно вставить её прямо в код:

jscript
let userName = "${username}";

onReady(function() {
print("Привет, " + userName + "!");
});

Но помни: для списков и сложных данных лучше использовать JSON-способ выше.

🛠️ Полный пример со всем вместе

Этот пример объединяет всё, что мы изучили: стили, HTML, кнопки, скрипты и приём данных с сервера.

css
basic

html
meta МойБлог

<script type="application/json" id="my-data"> { "posts": ${posts_json}, "user": "${username}" } </script><h1>Блог пользователя <span id="uname">...</span></h1> <div id="posts-box"></div>
btn Добавить запись
addPost()
endhtml

jscript
let blogData = {};

onReady(function() {
// Загружаем данные с сервера
let raw = gid("my-data").textContent;
blogData = JSON.parse(raw);

// Показываем имя
txt("uname", blogData.user);

// Рисуем посты
showPosts();
});

function showPosts() {
let htmlCode = "";

for(let i = 0; i < blogData.posts.length; i++) {
let post = blogData.posts[i];
htmlCode += "<div style='margin: 10px; padding: 10px; border: 1px solid #ccc;'>";
htmlCode += "<h3>" + post.title + "</h3>";
htmlCode += "<p>" + post.content + "</p>";
htmlCode += "</div>";
}

html("posts-box", htmlCode);
}

function addPost() {
let title = ask("Заголовок записи", "Новая запись");
let content = ask("Текст записи", "Напишите что-нибудь...");

// Добавляем новый пост
blogData.posts.push({
title: title,
content: content
});

// Перерисовываем список
showPosts();

print("Запись добавлена!");
}
